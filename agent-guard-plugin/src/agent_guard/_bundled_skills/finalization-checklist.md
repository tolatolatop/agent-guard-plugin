# Finalization Checklist

Before claiming completion:

1. Confirm the requested scope is complete.
2. List the concrete files changed.
3. Record the validation commands that were run and their results.
4. State any remaining risks, skipped checks, or blocked follow-up work.
5. Ensure no required edits remain pending.

Do not claim completion if validation is missing for a critical path or if known blockers remain unresolved.
